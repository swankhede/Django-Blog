# Django-Blog
Django-Blog is blogging website or webapp build using Django Framework 
in this website user can: 
<ul>
  <li>POST ARTICLES  </li>
   <li>READ ARTICLES OF OTHER USERS </li>
  <li>EDIT ARTICLES </li>
  <li>DELETE ARTICLES</li>
</ul>

# Technology Stack:
<ul>
  <li>HTML</li>
   <li>BOOTSTRAP</li>
  <li>DJANGO </li>
  <li>SQLITE</li>
  <li>HEROKU (HOSTING SERVER)</li>
</ul>
